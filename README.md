# Stock Price Analysis (Pfizer, Moderna, AstraZeneca)

## Overview
This project analyzes historical stock price data for major pharmaceutical companies
(Pfizer, Moderna, AstraZeneca) over multiple years to observe trends, volatility, and
price movements.

## Data Sources
- Yahoo Finance (via yfinance)

## Tools & Technologies
- Python
- pandas
- matplotlib
- mplfinance
- Jupyter Notebook

## Key Features
- Time-series analysis of stock prices
- Candlestick visualizations (OHLC)
- Data cleaning and formatting
- Comparison across multiple companies

## How to Run
```bash
pip install -r requirements.txt
jupyter notebook
